# Look Development — Phase 1.5 视觉基调确立

**这是 Color Script 的前置阶段，也是整个视觉系统的地基。**

## 目的

在进入具体色调分配之前，先确立：这个作品的视觉母题是什么？导演最关心的视觉主题是什么？观众看完这部作品后应该记住的视觉感受是什么？

## 输入

- `story_dna.json`（结构分析 + 情绪曲线）
- `director_intent.json`（Q1-Q6 答案 + additional_notes）

## Look Development 的五个维度

### 1. 视觉母题（Visual Motif）

这是最核心的问题：**这部作品在整个视觉体验上，导演最关心什么？**

不是问"这场戏是什么色调"，而是问"这部作品要建立什么视觉信念"。

格式：2-3句话，描述导演最核心的视觉关注点。

示例：
- "冷色调世界里的一抹暖色——那抹暖色是真实的还是幻觉？"
- "被困在秩序里的人，如何用身体语言打破画框的限制"
- "逆光剪影作为'真相被遮蔽'的视觉隐喻"

### 2. 参考图描述（Reference Direction）

不是上传图片（工具限制），而是**用语言描述导演脑中的视觉参考**。

格式：3-5个参考方向，每个用1-2句话描述。

参考维度：
- 摄影师/导演参考（"参考王家卫的抽帧色调和手持跟拍"）
- 光影参考（"参考维米尔窗边光的质感和冷暖对比"）
- 构图参考（"参考塔可夫斯基的远景长镜头和留白"）
- 色调参考（"参考《冷战》的低饱和暖黄色调"）
- 题材参考（"有《老友记》的空间感但色彩更消色"）

### 3. 色调关键词（3-5个）

整部作品的色彩关键词，不是具体的色值，是**感受性的色彩描述**。

示例：
- "压抑的冷蓝"（不是"6500K"）
- "被打破的暖黄"（不是"3200K"）
- "最终的消色"（代表某种放下或接受）

### 4. 摄影机哲学（Camera Philosophy）

对镜头运动的核心态度。回答："你相信镜头应该..."：

格式：2-3句话，描述导演对摄影机的基本哲学立场。

示例：
- "镜头应该像一只冷静的眼睛——它在场，但不介入。它记录，不评判。"
- "镜头应该是角色的心理延伸——当角色焦虑时，镜头轻微晃动；当角色清醒时，镜头稳定如锚。"
- "镜头应该在场——我们不是在看电影，我们是在透过一扇窗偷窥真实。"

### 5. 情绪-色彩基础映射

基于 Story DNA 的情绪曲线，给出一个**初步的全局情绪-色彩映射**。

格式：列出3个关键情绪节点和它们对应的色彩关键词。

示例：
| 情绪节点 | 色彩关键词 | 叙事含义 |
|---------|-----------|---------|
| Act1 压抑期 | 冷蓝+秩序感 | 建立世界的规则感 |
| Act2 挣扎期 | 冷暖交战中 | 真实与幻觉的拉扯 |
| Act3 结局期 | 消色或极端对比 | 某种放下或不可逆的选择 |

## 输出格式：lookdev.json

```json
{
  "story_title": "（来自 story_dna）",
  "visual_motif": "（2-3句话的核心视觉母题）",
  "reference_directions": [
    {
      "dimension": "摄影师参考",
      "description": "参考XXX的..."
    },
    {
      "dimension": "光影参考",
      "description": "参考XXX的..."
    }
  ],
  "color_keywords": ["压抑的冷蓝", "被打破的暖黄", "最终的消色"],
  "camera_philosophy": "（2-3句话描述摄影机哲学）",
  "emotion_color_mapping": [
    {
      "phase": "Act1 压抑期",
      "color_keywords": ["冷蓝", "秩序感"],
      "narrative_meaning": "建立世界的规则感"
    }
  ],
  "directors_notes": "（可选：导演补充的视觉笔记）"
}
```

## 与 Color Script 的关系

Look Development 是 Color Script 的**输入**，不是替代品。

Color Script 会引用 Look Development 的：
- `visual_motif` → 作为全局色彩叙事的核心依据
- `color_keywords` → 作为 beats 色调分配的参考词汇
- `emotion_color_mapping` → 作为色调-情绪对应的基础对照

## 自检清单

- [ ] `visual_motif` 回答了"观众应该记住什么视觉感受"
- [ ] `reference_directions` 有3个以上的维度
- [ ] `camera_philosophy` 回答了"镜头应该是什么角色"
- [ ] `emotion_color_mapping` 与 Story DNA 的三幕结构对应
